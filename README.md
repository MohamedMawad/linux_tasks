## Femto Shell Project:  
Simple Shell that supports echo for user input and exit command   


## Build Command:  
`gcc femtoShell.c -o femtoShell`  

## Run Command:
`./femtoShell`  


## Output Example:
```Femto_Shell_Prompt:# Hello   
You Said: Hello   
Femto_Shell_Prompt:# Test the shell  
You Said: Test the shell  
Femto_Shell_Prompt:# exit  
Good Bye :)```
